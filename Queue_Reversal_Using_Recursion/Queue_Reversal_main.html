Recursive Queue Reversal in C++

Reversing a queue using recursion eliminates the need for an additional data structure like a stack.

Code Implementation

We use recursion to reverse the queue. The key steps are:

  • Remove the front element using q.pop().
  • Call the recursive function on the remaining queue.
  • After the recursion stack unwinds, push the removed elements back using q.push(temp).

The function prototype is:

void reverseQueue(queue<int> &q, int &index, int size);

Test Case

Consider the following input queue:

Input: 1 2 3 4 5

Step-by-step process:

  1. Remove 1, call function recursively for [2, 3, 4, 5].
  2. Remove 2, call function recursively for [3, 4, 5].
  3. Remove 3, call function recursively for [4, 5].
  4. Remove 4, call function recursively for [5].
  5. Remove 5, base case reached.
  6. Now push back elements in reverse order: 5 → 4 → 3 → 2 → 1.
Output: 5 4 3 2 1

Time Complexity

The time complexity of this function is O(n), where n is the number of elements in the queue.
